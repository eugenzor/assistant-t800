# Suggestions Module

## Призначення

Модуль `suggestions` відповідає за підказки команд.

Він уже реалізований і в нормальному робочому процесі не потребує змін.

---

## Що він робить

- знаходить схожі команди через fuzzy matching;
- звертається до AI, якщо fuzzy не дає достатньо впевненої відповіді;
- перетворює природну мову на команду;
- повертає запропонований command input;
- дозволяє CLI запитати підтвердження.

Приклад:

```text
видали Аліса
```

може бути запропоновано як:

```text
remove Аліса
```

---

## Коли змінювати цей модуль

Тільки якщо потрібно:

- додати новий provider;
- змінити AI prompt;
- змінити threshold-и;
- виправити bug у suggestion pipeline.

Не треба переписувати цей модуль при додаванні звичайної команди.
