# Storage Module

## Призначення

Модуль `storage` відповідає за збереження та завантаження адресної книги.

Поточна реалізація використовує pickle.

---

## Шлях збереження

```text
.data/address_book.pkl
```

---

## Як працює

Storage використовується як context manager:

```python
with AssistantStorage() as contacts:
    ...
```

При вході:
- дані завантажуються з файлу;
- якщо файлу немає, створюється порожня адресна книга.

При виході:
- поточний стан зберігається на диск.

---

## Коли змінювати storage

Зазвичай не потрібно.

Можливі майбутні зміни:

- перехід на SQLite;
- export/import;
- backup;
- encrypted storage.

До цього часу storage краще не чіпати.
